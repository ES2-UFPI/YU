# YU
YU é uma aplicação que monitora dados do usuário e do ambiente para gerar recomendações personalizadas de bem-estar. A proposta é transformar informações do dia a dia em uma experiência visual simples e intuitiva.

# Funcionalidades

A aplicação coleta dados de localização, clima e uso do dispositivo, como tempo de tela e frequência de aplicativos. Com base nessas informações, ela analisa o comportamento do usuário e sugere atividades para melhorar sua rotina, além de apresentar um mascote virtual que reage às ações do usuário.

# Como Funciona

O sistema coleta dados, combina essas informações com os objetivos definidos pelo usuário e gera sugestões de forma dinâmica. O mascote virtual representa o estado atual do usuário, mostrando visualmente se seus hábitos estão positivos ou negativos.

# Objetivo

O objetivo é criar uma aplicação interativa e fácil de entender, que ajude o usuário a melhorar seus hábitos e manter uma rotina mais equilibrada, tanto no ambiente digital quanto no mundo real.
